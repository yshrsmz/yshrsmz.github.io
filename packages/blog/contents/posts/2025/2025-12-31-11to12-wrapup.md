## 仕事

### 気がついたら AI エージェントまわりとかオンボーディングとか

オンボーディング周りの自動化を、採用しているグループウェアの仕組みに乗っかっていい感じにするとか。年明けから人がたくさん来るらしいので頑張った。ぼちぼちいい具合になったのではなかろうか。やはりオンボーディングタスクはチケット管理システム使うのが安定です。

あとは色々なツールの検証をたくさんやっていた気がする。

## 個人プロジェクト

### 古いライブラリの近代化した

これ: [yshrsmz/historian: Custom Timber tree implementation that can save logs to SQLite](https://github.com/yshrsmz/historian)

まだ使われてそうな気配を察知したので、Android SDK バージョンの更新とか Kotlin 化とか色々やった。実に五年ぶりの更新。  
正直 Claude Code がなければやろうと思わなかったので、まじで生成 AI すごい。リリース自動化とか含めてシュッとできた。

### SinceTimer の Jetpack Compose 対応リリース

これも Claude Code と Codex 使って書き直した。自分でコードはほとんど書いていない。テーマ周りはあんまいい感じにならなかったので、テーマ生成ツールで生成したものを別で用意することになったけど、それ以外は概ねいい感じ。


### CI 充実させすぎて GitHub Actions の追加課金をすることになった

AI に任せて CI を充実させすぎた模様。仕事でやる分にはいいけど趣味はちょい自重しないとですね。どのタイミングで実行するかをちゃんと考えないといけない。だいたい30ドルくらい追加課金した気がする。
